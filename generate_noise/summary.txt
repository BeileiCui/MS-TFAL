# 1.To generate synthetic noise dataset, you have to first make sure you have the correct clean trainset like:
$./MSSTFAL/dataset/
├── endovis18
│   ├── train_clean
│   │   ├── seq_1
│       └── ├── labels
│           └── ├── framexxx.png
│           ├── left_frames
│           └── ├── framexxx.png
│       ├── seq_2
│           ...
│       └── seq_16

# 2. run below code to transfer multi class mask to multiple one class mask:
python generate_noise/multi_class_mask_to_multiple_one_class_mask.py

# 3. run below code to generate noisy labels, you may specify the parameters by your self:
python generate_noise/noisy_label_generation.py --ver 3 --noisy_ratio 30 --radius_low 40 --radius_up 50 --num_rays_low 4 --num_rays_up 8 --max_rotation_degree 30 --max_translate 0.2 --max_rescale 0.2

# 4. run below code to overlay the generated masks:
python generate_noise/overlay_sep_mask.py --ver 3

# 5. (Optional) you may get statistic for the noisy dataset:
python generate_noise/count_noisy_ratio.py --ver 3

# 6. (Optional) you may visualize the noisy dataset with below code:
python generate_noise/convert_to_RGB_mask.py --ver 3
